1. ~node.jsをインストールする。~
    - `node -v`
    - npmも使えるか確認`npm -v`
2. 初期化
    - `npm init -y`
3. `node ファイル名`
4. モジュールをインストールしたり？
    - `npm install --save-dev http`
5. サーバをいちいち落とさなくてもリロードされる。
    - `npm install -g nodemon`
    - `server.js`で保存をする。
